CheckBox[UICheckBox]

Icon

Description

UIScrollView is a subclass of UIWidget that implement check box of the touch screen.

The control provides methods to set the different states of the background image,check the selected tag of the state image and other attributes. By using the set method, you can specify a different apperance for the each checkbox state.
