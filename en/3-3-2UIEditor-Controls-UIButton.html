Button[UIButton]

Icon

Description

The UIButton class is a subclass of UIWidget, it implements a button on the touch screen.

Touch a button to intercept events and send the action message to the target object. The target and action set dervived from the UIWidget method. The control provides methods to set title, image, button and other apperance properties. By using the set method, you can specify a different apperance for each button state.
