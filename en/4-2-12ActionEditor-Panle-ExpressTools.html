Physical/Animation Mode Switch

     

 

Switch Physicak Mode and Animation Mode.

 

Rotated counterclockwise by 90 degrees

 

 

Rotated counterclockwise the selected widget by 90 degrees.

 

Rotated 90 degrees

 

 

Rotated the selected widget 90 degrees.

 

Align center point

 

 

Align all the selected widgets' center points- you should select more than one widget.

 

Rotate Skeleton Mode

 

 

The result of operating skeleton in the rendering area is rotating.

Move Skeleton Mode

 

 

The result of opreating skeleton in the rendering are is moving.

 

Zooming Mode

 

 

The result of opreating skeleton in the rendering are is zooming.

 

Display Skeleton Mode

 

Display the skeleton when selected the object, otherwise would not display. This switch just appears in Animation Mode.

 

Create Skeleton Mode

 

 

Create the skeleton when selected the object, otherwise would not create. This switch just appears in Physical Mode.

 

 

The two switches of display and create skeleton at the same position in the application, but they just appears in different situations.
