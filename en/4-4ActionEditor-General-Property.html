Skeleton Properties

General

Name

 

Set the name of skeleton. The pointer of the skeleton can be got by the name in the code.

 

Coordinate

 

Set skeleton's position. The origin of coordinate is the start point of drawing skeleton.

 

Frame Action

 

Set the name of the frame action of skeleton.

 

Rotation

 

Set rotational angle of the skeleton, property value is clockwise rotation degrees of the center axis of the control.

 

Display/Hide

 

Set display skeleton or not.

 

Rendering Resource

 

Set the picture of the skeleton's texture.

 

Parent Name

 

Display the parent name of the widget.

 

Zooming

 

Set skeleton scaling, zooming into the X-axis and Y-axis scaling. Values ​​can be positive or negative, flipping effect can be achieved when value is negative.

 

Transparency

 

Set the transparency of the widget. Value is between 0 and 255. 0 is transparent and 255 is opaque.

 

Locking

 

Lock the skeleton/texture, and it can not be edit if locked.

 

Color Mixing

 

This properties is setting color mixing of the widget. Click the drop-down menu color selection box will appear, after select the color the original widget will be mixed with the color rendering.
