Menu Bar

Introduction

 

Menu Bar is the set of common operations of editor. It mainly combined by File, Edit, View and Help. Use these four options to complete your operations


Functions

File(Alt+F)

New Project(Ctrl+N)

 

Create a new Action Project. If there has a project already opened in edit, it'll remind you save the project you are editing

 

Note:

1. The name of new project can not contains these notation: "?", "\", "*", "", "<", ">", "|". And avoid using Chinese character.

2. Do not save your new project on desktop if you are using Windows XP, any other path is OK.

3. Highly recommand you save your new project at any path as you want but System Safe Path

 

Open Project(Ctrl+O)

 

Open exist project

 

Save Project(Ctrl+S)

 

Save the project you are editing.

 

Save As(Ctrl+Shift+S)

 

Save the project you are editing at a new path.

 

Export Project

 

Export the selected canvas as the resource the program can use. Detail informations please check outUI Editor Project Export。

 

导入资源

导入文件

1.选择磁盘中的文件进行导入，支持多选择导入。

2.遇到同名文件时会提示同名，导入失败。

 

导入文件夹

1.选择磁盘中的文件夹进行导入。

2.遇到同名文件夹时会提示同名，导入失败。

 

Exit(Alt+F4)

 

Close the editor. If project changed, software will prompt you to save the project.

 

Edit(Alt+E)

Revoke

 

Revoke the last operation

 

Undo

 

Undo the operation revoked

 

View(Alt+V)

Properties

 

Open the object structure window

 

Resource

 

Open the resource

 

Output

 

Open the output window

 

Resource Edit Window

 

Open the resourec edit window

 

Undo the default layout of all the windows

 

Undo the default layout of the edit window

 

Help(Alt+H)

Check out Help(F1)

 

Check out online document

 

About Cocostudio

 

Open the cocostudio information window

 

New version check

 

Check out if there has new version and do upgrading

 

Langu(Alt+L)

English

 

Set the editor's language as English, then restart the editor.

 

Chinese

 

Set the editor's language as English, then restart the editor.
