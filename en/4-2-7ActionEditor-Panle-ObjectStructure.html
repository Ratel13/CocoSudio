Object-Relation Panel

Introduction

 

Object-Relation structure window display all the layers information of the current skeleton animation.

The window allows you hide the layer, lock the layer, change the zOrder of the layer and rename the layer etc.

 

Function

Display/Hide

 

1. Click the"eye"icon in the structure tree to set display or hide the widget.

2. Show hidden here and attributes zones show/hide is synchronous, they can be used to set the widget's Visible property.


Lock/Unlock

 

1.Click the "lock"icon in the structure tree to set lock or unlock the widget.

2. After widget locked all the operations of widget will be locked.

 

Hierarchy Adjust

Adjust hierarchy zOrder will change the display order. In object structure window adjust zOrder will affect all frames' zOrder. The one has large zOrder in the upside, all the top-level zorder are max. You can adjust the hierarchy of single or multiple selection.

 

Move current node up to a layer.
Move current node down to next layer.
Set current node to the top.
Set current node to the buttom.

 

Create a frame sequence animation

 

You can select one or more resources in the window then hold on left mouse button to drag them to this layer. The layer will create the same number of frame automatically. The display obbject of every frame is the corresponding resource, and the interval between every frame are 5 seconds.

 

Multiple Hierarchy

 

1. Select a layer and hold on Shift to choose another.

2. Hold on Ctrl and select layers.

3. Hold on Ctrl+a and select all the layers.

 

Content Menu

Add Node

Add a new node at current node.

 

Copy Node

 

Copy current node and add it to the end of the node list.

 

Delete Node

 

Delete the current node.
