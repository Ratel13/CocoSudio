TextArea[UITextArea]

Icon

Description

UITextArea is a subclass of UIWidget that implement text area of the touch screen.

The widget provides methods to set the text area of the display width and height, font size, font, alignment, text content, whether scaling and other attributes.

Feature

Width

 

Set the widget display width.

 

Height

 

Set the widget display height.

 

Font size

 

Display font size.

 

font

 

Now supports only Arial, bold, Microsoft yahei.

 

Align horizontal

 

Set the horizontal alignment of the widget:

1.HT_Left: Text align left

2.HT_Center: Text align center

3.HT_Right: Text align right

 

Align vertical

 

Set the vertical alignment of the widget:

1.VT_Top: Text align top

2.VT_Center: Text align center

3.VT_Bottom: Text align bottom

 

Text

 

The text display.

 

Text scale

 

Controls whether the text is scaled.
