Introduction

CocoStudio Introduction

CocoStudio is a suit of free mobile game development tool set based on cocos2d-x.

Unlike other tools with a full set of features integrated, Cocostudio divides these tools into four, including UI editor, animation editor, scene editor and data editor. These four editors are tailored for UI artist, action artist, game designer and game system designer.

Our purpose is maximizing the productivity of every kind of roles in game development. Let the artists to focus more on aesthetics, let the game designers to focus more on game play without having to wait for the programmer's work done. In the end, we could promote the cycle of game iteration. With more iteration cycle and the game will be more successful. At the same time, for dividing tool sets, each role can pick up each tool easily. CocoStudio aims to help mobile game developers to develop more awesome games.

Install & Uninstall

Software environment requirement

  1. Windows XP / Windows 7 / Windows 8
  2. Runtime .Net Framework 4.0
  3. Visual C++ 2010 Redistributable
  4. Visual C++ 2012 Redistributable

If your computer don't meet above requirement, you to refer to Microsoft's website to download them freely.Net Framework 4.0And Visual C++ 2012 Redistributable。

Hardware environment requirement

  1. CPU：2G以上.
  2. Memory：1G以上.
  3. Hard disk：100M以上(安装所需大小).

Install instructions

  1. Double click to open the installation package.
  2. Read and accept the Software Usage License.
  3. Click Next to continue.
  4. Choose your installation directory.
  5. Finish installation.

Update

 

CocoStudio will check whether there is a new version available automatically when the program is launched.

Update will uninstall your old copy and replace it with the new one. It doesn't do any harm to your existing projects.
