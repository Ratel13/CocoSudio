Introduction

CocoStudio Introduction

CocoStudio is a free suite of mobile game development tools that helps mobile game developers create awesome games.

<<<<<<< HEAD

CocoStudio contains a tool set of four editors: the user interface (UI) editor, animation editor, scene editor and data editor. The four editors are tailored for the roles of UI artist, action artist, game designer and game system designer.

CocoStudio maximizes the productivity of game development and aesthetics. By using CocoStudio, game designers can focus more on game design without having to wait for the completion of the programmer's work. CocoStudio allows for more iterative cycles of game development. With more productive creative cycles, the game development will be more successful. At the same time, with separate tool sets, each role can easily work with each tool.

<<<<<<< HEAD

You can refer to the CocoStudio website at cocos2d-x.org to download the newest version, which includes the CocoStudio installation package and corresponding parsers.
