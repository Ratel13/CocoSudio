Canvas list panel

Introduction

 

1.Manage the canvas for the project.

2.Use multi canvas to save or display the contents of different rendering area.


Functions

New Canvas

 

1.Use right-click menu to create multiple blank canvas.

2.Within the same project, the canvas size is consistent.

 

Switch canvas

 

1.Canvas can smooth switching within the same project.

2.Resource files can be reused between canvas.

 

Copy controls between canvas

 

Support copy the contents of one canvas to another canvas.

 

Right-click menu

New Canvas

 

Use right-click menu to create multiple blank canvas. Default canvas name (project name + “ _ ” + number).

 

Rename

 

Name can not be empty, can not be the same name already exists, the following symbols can not be included: "？"、"、"、"╲"、"*"、"""、"""、"<"、">"、"|" ; Shortcut F2.

 

Delete

 

Delete the selected canvas. But the last canvas can not be delete. shortcut Delete.

 

This deletion will not be able to recover, there will be a confirmation prompt.

Copy

 

copy the selected canvas, shortcut Ctrl +C。

 

paste

 

paste the selected canvas, shortcut Ctrl +V。
