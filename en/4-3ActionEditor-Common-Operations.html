General operations of action editor

Create skeleton

Animation Examples

 

Needed Mode

  1. Physical Mode
  2. Create Skeleton

Procedure

 

1. Click the left mouse button in the rendering area to start drawing skeletons.

2. Release the mouse to complete the draw. A anchor will be displayed if draw too short.

 

Modify Skeleton

Animation Examples

 

 

 

Needed Mode

  1. Physical Mode
  2. Stop create skeleton

Operation

Adjust skeleton's direction
Click the Rotate button in quick tool, drag the needed skeletons to be adjusted to adjust the direction of the bone.
Adjust the pivot position
Click the pan buttons in the quick tool, drag the skeletal that needed to be adjusted, then the position of the skeleton can be adjusted.
Adjust the length of the skeleton
Click the Zoom In button in quick tool, drag the skeletal that needed to be adjusted, then the length of the skeleton can be adjusted.

Add Texture

Animation Example

 

Needed Mode

  1. Physical Mode
  2. Stop to create skeleton

Procedure

 

Solution A: Select skeleton, drag the picture from resource panel to the rendering resourec properites box of properties panel.

Solution B: Drag texture from resource panel to rendering panel.

 

Note: Solution A will be binded to skeleton directly, and won't be changed.

Bind to bone

Animation Example

Needed Mode

  1. Physical Mode
  2. Stop to create skeleton

Procedure

 

1. Select texture in rendering panel and then select content menu"bind to skeleton(ALT+F)".

2. Select the skeleton in the rendering panel, which is need to be bind.

3. Software notification: "picture will be bind to skeleton, continue?"Select"Yes".

 

Bind parent relationship

Animation Examples

Needed Mode

  1. Physical Mode
  2. Stop to create skeleton

Procedure

 

1. Select the skeleton in the rendering panel, which is about to be selected as child.

2. Select content menu"bind to parent(ALT+P)".

3. Slect the skeleton, which is about to be parent.

 

 

Note: One skeleton has only one parent, but it can has many children.

 

Delete Selected

Animation Examples

 

Needed Mode

  1. Physical Mode
  2. Stop create skeleton

Procedure

 

1. Select the skeleton in the rendering panel, which is about to be deleted.

2. Select content ment "Delete Selected(DELETE)"then the object can be deleted.

 

 

Note: All the children will be removed when deleting, if just delete the current skeleton, parent relationship need to be removed.

 

Remove parent relationship

Animation Examples

 

Needed Mode

  1. Physical Mode
  2. Stop to create skeleton

Procedure

 

1. Select the skeleton in the rendering panel, which is about to be deleted.

2. Select content ment "remove parent relationship(ALT+D)"then binding relationship can be removed.
