Loading bar[UILoadingBar]

Icon

Description

UILoadingBar is a subclass of UIWidget that implement progress bar of the touch screen.

The widget provides methods to set value of the progress bar, progress bar orientation, image resources and other attributes.

Type

 

Set the progress bar orientation, Left_TO_Right(leaf to right) and Right_To_Left(from right to left) and Left_To_Right by default.

Configure the image resources (png, jpg), control the default display.
