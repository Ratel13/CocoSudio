Launch

 

On default, after you installed the CocoStudio, it will generate a shortcut on your desktop( see the above picture). Click on the shortcut to launch CocoStudio. If the shortcut is missing or invalid, you can go to the installation directory of CocoStudio and click the Cocostudio.exe to finish launching.

 

Start Screen

 

The above picture is the screen after launch. It has LOGO area, editor launch button area and settings button area. The upper half is CocoStudio's LOGO, and the lower half from left to right each is Action Editor, UI Editor, Scene Editor and Data Editor. On the bottom right, there is a settings button.If you click the button, you can get the following result:

 

 

Settings screen includes Settings and Update two parts.

Setting

Language

 

Currently CocoStudio support Chinese and English. You can choose your preference from the drop down menu item.

 

Render Mode

 

Cocostudio supports hardware and software rendering modes. Hardware render mode is much more fast, while software render mode is more stable.Your can tweak it yourself.

 

Changing render mode is not always valid, it requires system environment support. You may find the display area is black when Hardware render mode is activated, now you should change it to Software render mode.
