Scene Component Gerneral Properties

 

Scene Component Gerneral Properties are consisted of common properties and special properties.  Each component has some common properties and one or more special properties. Components will have common properties when added , and all components can have a ComAttrubite property and audio.

 

常规属性

Tag

 

Set the Tag of component, and we can get this compent by it.

 

ZOrder

 

Set the display hierarchy of this component , and the program will rendering components following the value of ZOrder.If the value if bigger , the component will be closer to monitor.

 

Position

 

Set the position of component's anchor point. The default anchor point is the center of component. The origin is the left down corner of the canvas.

 

Alias

 

Component's name, which can let us get component in programs.

 

Rotation

 

Set the property of component's rotation. The value of Rotation is the clockwise degree of component's rotating by anchor point.

 

Visible

 

Controlling the visibility of component. Users can hide some components at first and show them in programming.No matter if the component hide or not, it will be load into cache.

 

Scale

 

Set the scale value of component, and it consist of scaleX and scaleY. The value can be a negtive number or a positive number. If it is a negtive value, the component can filp.
