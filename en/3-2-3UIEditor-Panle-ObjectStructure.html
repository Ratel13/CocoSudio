Object Struct Panel

Overview

 

Object Struct Panel manage all control in the UI interface, including child-parent relationship management and hierarchy management. The root object of the Object Struct Panel is a Panel.

 

 

Functionality

Show/Hide

=======

对象结构面板

简介

 

对象结构面板提供对UI界面中的所有的控件的管理功能，包括父子关系管理和层级关系管理。结构树的根节点是一个Panel。

Layerout align

Center Alignment Make the selected controls center align.
Left Alignment Make the selected controls left align.
Right Alignment Make the selected controls right align.
Top Alignment Make the selected controls top align.
Bottom Alignment Make the selected controls bottom align.
Horizontal Alignment Make the selected controls horizontal align.
Vertical Alignment Make the selected controls vertical align.
Origin Alignment Make the selected controls origin align.

 

  1. If no control is selected or only one control is selected, the menu item will be disabled.
  2. Alignment will ignore all sub-controls of the selected control, because the sub-control is changing according to its parent.
  3. Theory
    1. Compute the whole common rectangle of all controls to prepare for aligning.
    2. Convert each control's four point to world coordination.
    3. Calculate each control's four edge, left side, right side, top side and bottom side.
    4. Calculate the offset. For example, control's left side coordination - all controls' common rectangle's left side = the offset
    5. Subtract the offset from control's coordination to achieve alignment.
