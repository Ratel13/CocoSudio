Object Struct Panel

Overview

 

Object Struct Panel manage all widget in the UI interface, including child-parent relationship management and hierarchy management. The root object of the Object Struct Panel is a Panel.

 

 

Functionality

Show/Hide

 

1. Click the Eye icon on the object tree to toggle visibility of the widget.

2. We can use this functionality to set widget's visible property.


Lock/Unlock

 

1.Click the Circle icon to lock or unlock widget.

2.When the widget is locked, all the operations on it will be disabled.

 

Hierarchy Adjustment

 

1. You can adjust the hierarchy of widgets by dragging.

2. As you see in the picture: When you drag the ImageView widget to the next panel, there will be a blue line between the two panel. If you release your mouse, then the ImageView will be putted between the two panels. If you drop the ImageView on top of a panel, then there will be a blue circle around the panel, when you release the mouse, the ImageView will be added as a subset of the panel.

Layerout align

Center Alignment Make the selected widgets center align.
Left Alignment Make the selected widgets left align.
Right Alignment Make the selected widgets right align.
Top Alignment Make the selected widgets top align.
Bottom Alignment Make the selected widgets bottom align.
Horizontal Alignment Make the selected widgets horizontal align.
Vertical Alignment Make the selected widgets vertical align.
Origin Alignment Make the selected widgets origin align.

 

  1. If no widget is selected or only one widget is selected, the menu item will be disabled.
  2. Alignment will ignore all sub-widgets of the selected widget, because the sub-widget is changing according to its parent.
  3. Theory
    1. Compute the whole common rectangle of all widgets to prepare for aligning.
    2. Convert each widget's four point to world coordination.
    3. Calculate each widget's four edge, left side, right side, top side and bottom side.
    4. Calculate the offset. For example, widget's left side coordination - all widgets' common rectangle's left side = the offset
    5. Subtract the offset from widget's coordination to achieve alignment.

Delete Object

 

Delete the selected object

 

 

1. If no widget is selected, this menu item will be disabled.

2. Support multiple deletion.

 

Copy

 

Make a copy of the selected widget.

 

 

If there is no widget to be selected, the menu item will be disabled.

 

Paste

 

Paste the pre copied widget.

 

 

If there is no widget be pre copied, then the menu item will be disabled.

 

Edit Animation

Only works when you switch to Animatin Model from the menu bar.
