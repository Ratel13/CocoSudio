Rendering Panel

Introduction

Rendering window provide a panel to draw skeleton animation, and it provides preview whenever you want.

 

Functions

Select Region

 

Use mouse select widget, adjust the whole or adjust the single skeleton.


Adjust Skeleton/Textures

 

Description:

  • At 1, 2, 4 and 5 you can use mouse to adjust the size of the widget, and do the scaling operation to it.
  • At 3 you can do rotating operation to widget.
  • 6 is the anchor point.

Content Menu

Move the selected up to the top.

 

Move the selected skeletons or textures layer to the top. The image on the top can cover other bones or textures, which located under it.

 

Move the selected

 

 

Move the selected skeletons or textures up a layer.

 

Move the selected down to buttom

 

 

Move the selected skeleton or texture down to the next layer.

 

Cancel the selected

 

Cancel the selected skeletons and textures.

 

Binding parent

 

 

Select the parent skeleto for the selected skeletal. This option is only has effect on parent skeleton, which haven't binded.

 

Remove the parent

 

Remove the parent-child relationship for the selected skeletons. This option is only has effect on binded skeletons.

 

Bind to skeleton

 

Bind skeleton for the selected texture. This option is only has effect on the texture, which hasn't binded to skeleton.
