Render Panel

Overview

 

Render Panel provides a rendering control to support real-time preview.

 

Functionality

Select

 

You can use your mouse to select the whole control and do copy, delete operation on it.


Adjust View

 

Details:

  • At the 1,2,4,5 point, you can use your mouse to adjust control's size . You can also scale the control(With the Shift key be pressed , you can achieve a Uniform-Scale)
  • At the 3 point, you can rotate the control.
  • Point 6 is a anchor point.

 

Context Menu

Add Object

Button Add a button to a control as a sub-control.
CheckBox Add a CheckBox to a control as a sub-control.
Image Add a Image to a control as a sub-control.
Label Add a text field to a control as a sub-control.
AtlasLabel Add a numeric label to a control as a sub-control.
BMFontLabel Add a user-defined font to a control as a sub-control.
LoadingBar Add a loading bar to a control as a sub-control.
Panel Add a layer lick container to a control as a sub-control.
ScrollView Add a scroll view to a control as a sub-control.
Slider Add a slider to a control as a sub-control.
TextArea Add a text area to a control as a sub-control.
TextButton Add a text button to a control as a sub-control.
TextField Add a text field to a control as a sub-control.
DragPanel Add a dragable panel to a control as a sub-control.

 

When you select a control and add a control through context menu, it will add a control to your current selected control. If you don't select a control before you fire a context menu to add a control, it will be added as a root control's sub-control.

Layout align

Center Alignment Make the selected controls center align.
Left Alignment Make the selected controls left align.
Right Alignment Make the selected controls right align.
Top Alignment Make the selected controls top align.
Bottom Alignment Make the selected controls bottom align.
Horizontal Alignment Make the selected controls horizontal align.
Vertical Alignment Make the selected controls vertical align.
Origin Alignment Make the selected controls origin align.

Delete Object

 

Delete the selected controls.

 

  1. If no control is selected, then the menu item will be disabled.
  2. Support multiple selection.

Copy

Make a copy of the selected controls.

 

 

If no control is selected, then the menu item will be disabled.

 

Paste

Paste the pre-copied controls.

 

 

If there are no controls are copied before, then the menu item will also be disabled.

 

Edit Animation

Edit animation of the selected control.

 

 

You can't edit animation of control until you switch to the animation model.
