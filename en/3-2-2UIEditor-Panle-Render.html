Render Panel

Overview

 

The Render panel provides a rendering control to support real-time previewing functionality.

 

Functionality

Select

 

Use your mouse to select the entire control, then copy or delete the control.


Adjust View

 

Details:

  • At the corners or the 1,2,4, or 5 points of reference as labelled above, you can use your mouse to adjust the size of the control. You can also scale the control by simultaneously pressing the Shift key and clicking on the control to modify the control to be uniform to scale.
  • By selecting the point of reference as shown in the graphic as point 3, you can rotate the control.
  • Note that point 6 is an anchor point.

 

Context Menu

Add Object

Button Adds a button to a control as a sub control.
CheckBox Adds a checkbox to a control as a sub control.
Image Adds an image to a control as a sub control.
Label Adds a text field to a control as a sub control.
AtlasLabel Adds a numeric label to a control as a sub control.
BMFontLabel Adds a user-defined font to a control as a sub control.
LoadingBar Adds a loading bar to a control as a sub control.
Panel Adds a layer lock container to a control as a sub control.
ScrollView Adds a scroll view to a control as a sub control.
Slider Adds a slider to a control as a sub control.
TextArea Adds a text area to a control as a sub control.
TextButton Adds a text button to a control as a sub control.
TextField Adds a text field to a control as a sub control.
DragPanel Adds a dragable panel to a control as a sub control.

 

When you select a control and add a control through the context menu, CocoStudio will add a control to your currently selected control. If you don't select a control before you fire a context menu to add a control, CocoStudio will add the control as a root control's sub control.

Layout Alignment

Center Alignment Center aligns the selected controls.
Left Alignment Left aligns the selected controls.
Right Alignment Right aligns the selected controls.
Top Alignment Top aligns the selected controls.
Bottom Alignment Bottom aligns the selected controls.
Horizontal Alignment Horizontally aligns the selected controls.
Vertical Alignment Vertically aligns the selected controls.
Original Alignment Revert the selected controls back to the original alignment.

Delete Object

 

Deletes the selected controls.

 

  1. If no control is selected, then the menu item will be disabled.
  2. Supports multiple selection.

Copy

Makes a copy of the selected controls.

 

 

If no control is selected, then the menu item will be disabled.

 

Paste

Pastes the pre-copied controls.

 

 

If no controls were copied before, then the menu item will also be disabled.

 

Edit Animation

Edits animation of the selected control.

 

 

You must switch to the animation model first in order to edit the animation of the control.
