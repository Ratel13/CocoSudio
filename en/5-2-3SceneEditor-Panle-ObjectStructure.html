Object Structure Panle

Introduction

 

1. The root of the structure tree is a Node that represents a game object。

2. The added object in structure tree has a Corresponding object in Render Area.

 

Feature

Show/Hide

 

1. Click the 'eye' icon to set the widget Show or Hide.

2. This Show/Hide buttom used to set widget's Visible proporty and it is synchronized with proporty panle.


Lock/Unlock

 

1. Click the 'lock' icon to set the widget Lock/Unlock.

2. Control locked will shield all operations.

 

Layers Adjustment

 

Adjust the level of the widget by Drag the mouse through the structure tree.

 

Right Click Menu

Add Object

 

Adding an object to the canvas.

 

When you selected a scene and right click add it to another object as subset, please note that: if you selected nothing as target object, the scene will be added to root object as subset.

 

Delete Object

Delete the selected object

1.If no object or only one object is selected, this menu item is masked.

2.If you delete a object which contains a subset, the subset will be deleted.

 

Add Components

Armature Component Armature animation component is a rendering component which is used to rendering Armature animations. If you mount Armature animation component, empty object can be converted into an animation object.
Map Component Map component is a rendering component which is used to rendering tmx format map file. If you mount the map component, empty object can be converted into a map object.
UI Component UI component is a rendering component which is used to rendering UI controls. If you mount the UI component, empty object can be converted into a UI object.
Particle Component Particle component is a rendering component which is used to rendering pliat format Particle file. If you mount the Particle component, empty object can be converted into a Particle object.
Custom Property Custom Property Component is used to load data from json file.
Audio Audio Component is used to play mp3 and wav fomat audio file.
Sprite Sprite component is a rendering component which is used to rendering png format image file.

 

1.An empty object can only mount one rendering component.

2.An empty object can only add an audio component and a Custom Property Component.

 

Delete Component

 

Armature Component Remove Armature Component
Map Component Remove Map Component
UI Component Remove UI Component
Particle Component Remove Particle Component
Custom Property Remove Custom Property Component
Audio Component Remove Audio Component
Sprite Component Remove Sprite Component

 

If selected node does not contain a component, the component corresponding menu item will be shielded.

 

Copy

1. If no object is selected, the menu item will be masked

2. Multi-selection Copy Support(Ctrl+C).

Paste

1.If the object is not copied, this menu item is masked.

2.Pasting position varies according to the mouse position.

3.When using Shortcut Ctrl + V paste, the current position is the pasted object replication.
