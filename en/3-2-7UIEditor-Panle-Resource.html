Resource Panel

Introduction

 

Editing UI animations in UIEditor need to switch to Animation mode, after complete, switch to Normal mode to do UI layout.

 

Functions

Add file

 

1.Select files from disk to import. Support multi-file import.

2.Files with exist name will not be imported.

 

Add Folder

 

1.Select folder from disk to import

2.Folder with exist name will not be imported.

 

Refresh Resource

 

1.Use to refresh the list of resources in project when user manually delete or add file or folder under resources folder.

 

Search

 

Filter out resources which name match the user's input.

 

Preview Resource

 

1.Preview the selected resources.

2.Mouse hover preview.

3.Double-click to preview with computer's picture viewer.

 

Drag to config resources

 

1.Drag pipe2.png in resource area to the file properties box to complete the configuration of the control file attributes.

 

Right-click menu

Open in Explorer

1.Open select file with Windows Explorer.

Rename

1.Rename resource name, Not Support Plist rename.

2.Not Support illegal character.

3.Shortcuts F2.

 

Delete

 

Delete selected resource.

 

Import

Import file

1.Select files from disk to import. Support multi-file import.

2.Files with exist name will not be imported.

 

Import folder

1.Select folder from disk to import.

2.Folder with exist name will not be imported.

 

Export Image

 

Export plist picture as divided picture.

 

Refresh Resource
