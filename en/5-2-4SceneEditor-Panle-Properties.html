属性面板

简介

 

1. When you select an object, the properties window will show all the properties of the object.

2. Modify any proporty can change the status of the Node in rendering area immediately.

Feature

Search

1. Description of the currently selected object is shown in the top.

2. Search: Filtered matching property according to the user input text.

General Properties

Tag Find the identify of Node code. (when added a new Node, tag will increase 1 automatically).
ZOrder Adjust rendering order or occlusion of the Node.
Position Adjust the position of the widget.
Alias Display the alias of node in the object structure panel.
Rotation Adjust the rotation angle of widget.
visible Whether to display a subset of Node.
Scale Zoom the widget by horizontal or vertical.
