Actions List Panel

Introduction

 

The panel provide actions management of the project.


Function

Change the name of Animation

 

If you want to modify the name of the action, use the F2 key to open the state modifying the name of the animation.

 

Content Menu

Add Animation

 

Add a new animation at current position.

 

Copy Animation

 

Copy a current selected animation and add it to the end of the list.

 

Delete Animation

 

Delete the current selected animation.
