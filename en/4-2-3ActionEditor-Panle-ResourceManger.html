Resource Manager Panel

Introduction

 

In Resource Editor window you can edit the collision zone and anchor information of the selected skeleton's current using resources.


Introduction

Accepts collision

Check box is used to set whether to create collision zone when the current mouse Box is checked. If checked, hold down the left mouse button to draw the textures collision zone.

 

Collision Zone

Hold down the left mouse button in the Explorer panel, and drag the picture to the draw collision zone. The collision zone was translucent rectangular area.

 

Zoom In

Plus button is used to control the zooming injianh operation of resource canvas.

 

Zoom Out

Minus button is used to control the zooming injianh operation of resource canvas.

 

Anchor Point

Setting the anchor point of skeleton or textures. Set the point as the axis when choose skeleton or textures.

 

Content Memu

Cancel

Cancel a collision area

 

Change Color

Change the color in collision area.

 

Change Background

Choose pictures in disk and set one of them as background.
