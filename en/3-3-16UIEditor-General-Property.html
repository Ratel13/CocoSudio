3x3 grids

3x3 grids is to use a small picture to draw the large areas picture without distortion techniques.

When drawing will follow the rules below:

 

a. maintain four horn parts without distortion

b. uniaxial tensile four edges (i.e. the edges between any two corners, such as top, only transverse tensile)

c. (i.e. the 3x3 grids middle part, horizontal, vertical, simultaneously stretched, PS: stretching the proportions are not necessarily the same)

 

CocoStudio of four parameters:

 

"CapInsetsX"、"CapInsetsY"  is a pair of parameters, it can determine our center coordinates of the lower left corner (Cocos2d-x is the lower left corner as the origin);

"CapInsetsWidth"、"CapInsetsHeight" is determining the size of center；

 

 

Note ：

Widget width greater then 2*X, widget height greater than 2*Y, or prone to map the wrong.

Set 3x3 grids attribute value should be scaling properties are set to 1, otherwise the picture will still appear distorted.

 

Common properties

Interactive

 

Control widget accepts user events or not, if the widget to accept user event (such as a button), you need to check.

 

Show/Hide

 

Controls whether the control is displayed. Lets start widget does not appear, the latter through the code and then displayed. But neither displayed or not, the widget will be initialized to load into memory.

 

Flip vertical

 

Checked, the widget will display vertical image.

 

Flip horizontal

 

Checked, the widget will display horizontal image.

 

Tag

 

Set the widget's tag, can obtain the widget through the tag values in the program.

 

Rotation

 

Set the widget's rotation property, property values for the widget center to anchor clockwise rotation.

 

Scale

 

Set the widget's scaling, divied into X axis scaling and Y axis scaling. Values can be positive or negative, when the negative effect can be achieved when the flip.

 

Position

 

Set the widget's anchor location, the default value is the widget center. Coordinate origin is the bottom left corner.

 

Anchor point

 

Set the anchor position, the default is that the center of 0.5/0.5 .

 

Render layer

 

Set the widget's rendering order. Smaller values are first rendering, but may be rendered after coverage.

 

Transparency

 

Set the widget's transparency. Ranges from 0 to 255. 0 for the pure transparent, 255 is opaque.

 

Color blend

 

Set the widget's blend color. Click the drop-down menu will appear after color selection box, select color after mixing the original widget and the color rendering.
