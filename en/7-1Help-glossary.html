术语表

cocostudio

 

  CocoStudio is a toolkit for cocos2d-x multi-platform games. It consisting of a UI Editor, an Animation Editor, a Scene Editor and a Data Editor.

 

Plist File

 

 Plist File always saves the settings of users', and it alse saves the binding information. This function was served by eldery Mac OS's resources branch.

For CocoStudio, some setting configure file is using plist file to save.

 

Frame

 

Frame means the Frame Animation, which is using a series of images to play a animation. Frame animation is an efficient skill to create a special effects, and it is a widespread skill in games.

 

DragoneBones

 

 DragonBones is an open source tool and framework for 2D-skeleton-animation. It contains a skeleton animation edit panel and ActionScript framework for skeleton animation, which is base on Flash Pro. In order to improve the efficiency and the utilization of current resources,CocoStudio's Animation Editor support the importing of DragonBones projects.

 

Particle

 

 Particles is a kind of interface element which have the attributes of appearance,position,speed,accelerated speed and life circle, and they produce run and die following some rule.
