Menu Bar

Overview

 

Menu bar contains all of your daily used operations. It contains File, Edit, View, Help and Language totally in five parts.


Functionality

File(ALT + F)

New Project(Ctrl + N)

 

Create a new UI project. If a project is already opened, it will prompt to save the current project. When you set up a new project, you should specify the project name and location.

 

 

Note:

1：Project name can't contains the following symbols : "?", "`", "\", "*", "<", ">","|""

2. When CocoStudio is running on Windows XP, you'd better not put the project on your desktop.

3. You'd better not put project under system directory.

 

Open Project(Ctrl+o)

 

Open the existing project.

 

Close Project

 

Close the opening project.

 

Save Project(Ctrl-S)

 

Save the opening project.

 

Save as(Ctrl-Shift+S)

 

Save the project to another directory.

 

Export project

 

Export the selected canvas. Please refer to UI editor project export。

 

Import template

 

Import the existing project.

 

Note:

1. It only supports importing the json files which are exported by UI editor.

2. Before you import json files, it will import all the resources which are under the same directory to CocoStudio. When it encounters two directories with the same name, it will compare the two directory and copy files smartly.

Import Resource

Import file

1. Import a file from your hard disk. It supports multiple imports.

2. When it encounters a same file, importing will be failed.

 

Import directory

1. Import a directory from your hard disk.

2. When it encounters a same directory, importing will be failed.

 

Open CCB file

 

Import CocosBuilder project

 

Exit(Alt+F4)

 

Close editor.If the project is modified, it will prompt user to save the project.

 

Edit(ALT+E)

Undo(Ctrl-Z)

 

Undo the last operation.

 

Redo(ctrl+y)

 

Redo the last Undo operation.

 

Canvas Scale

 

Do scaling operations on canvas.

 

View(ALT+V)

Object Structure

 

Open Object Structure Windows

 

Property

 

Open property windows.

 

Resource

 

Open resource windows.

 

Canvas List

 

Open canvas list windows.

 

Output

 

Open the Output windows.

 

Start page windows.

 

Open the start page window of UI editor.

 

Reset windows layout

 

Reset CocoStudio's window layout to it's default value.

 

Help(ALT+H)

View Help(F1)

 

View the online help documentation.

 

About CocoStudio.

 

Open CocoStudio about window.

 

Check New Version

 

Check whether there is a new version existing.

 

Language(ALT+L)

English

 

Set English as the editor's default language.It will take effect only after relaunch your editor.

 

Chinese

 

Set Chinese as the editor's default language. It will take effect only after relaunch your editor.
